# Memory Game

Игра на память с карточками, написанная на Python с использованием PyQt5.
Игрок должен находить пары одинаковых изображений.

**Основные возможности:**
1. Геймплей: Поиск пар карточек за ограниченное количество ходов
2. Уровни сложности: 4 базовых уровня + бесконечные комбинации
3. Звук и музыка: Фоновая музыка и звуковые эффекты
4. Сохранение прогресса: Автоматическое сохранение рекорда и уровня
5. Настройки: Управление звуком и музыкой в реальном времени

**Правила игры:**
1. На поле 16 карточек (8 пар)
2. Кликните по карточке, чтобы перевернуть
3. Найдите две одинаковые карточки подряд
4. Каждая неудачная попытка уменьшает количество ходов
5. При успешном прохождении открывается следующий уровень

**Требования**
Python 3.8 или выше
PyQt5

**Запуск:** 
python main.py

**Возможности:**
1. Запуск со стандартными ресурсами: python main.py
2. Запуск с пользовательскими ресурсами:

Пример кода:
    custom_paths = {
        'ui': './my_ui/game.ui',
        'images': './my_images/',
        'music': './sounds/my_music.ogg' }
    game = MemoryGame(custom_paths=custom_paths)

**Технические требования**
Необходимые ресурсы:
* 4 папки с изображениями (images_1 - images_4)
* Минимум 8 изображений в каждой папке
* Звуковые файлы (WAV): flip.wav, match.wav, win.wav, lose.wav
* Музыкальный файл (OGG): music.ogg
Поддерживаемые форматы:
* Изображения: PNG, JPG, JPEG
* Звуки: WAV
* Музыка: OGG, MP3
